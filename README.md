# Wine Quality Prediction using Support Vector Machine (SVM)

This project predicts wine quality using Support Vector Machine.
Both Multiclass Classification and Binary Classification were implemented on the Wine Quality dataset.

## Project Workflow
1. Data Loading
2. Data Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Scaling
5. Multiclass Classification using SVM
6. Binary Classification using SVM
7. Model Evaluation using Confusion Matrix and Classification Report

## Files in Repository
- machine_learning_svm_project.ipynb : Main notebook
- winequality-white.csv : Dataset
